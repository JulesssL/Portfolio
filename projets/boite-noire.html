Description

Pour mon premier projet de Partiels (après 4 mois d’études), nous avons dû réaliser toute la partie communication digitale & Marketing (Site Web, Design, Stratégie Marketing) pour un restaurant fictif à Lille.

Pour ce faire, nous avions du lundi au jeudi pour le réaliser par groupe de 4. Je m’occupais donc dans ce groupe de la partie développement Web pour le site dans cette entreprise fictive.

Objectifs

Réaliser un site web Front présentant le Restaurant.
Faire un site Responsive clair.

Apports

Création de la maquette grâce à Figma.
Réalisation des pages web (HTML - CSS - JS).
Gérer les liens et l’harmonie entre les pages.
Mettre en ligne le site via un hébergeur et un FTP.
