Technologies utilisées

Description

Pour ce projet, nous avions 3 jours pour analyser le site actuel de Airbnb, remarquer quels sont les points à améliorer, créer une maquette pour les pages importantes et les développer en duo.

Le délai étant assez serré, nous nous sommes concentré sur l'essentiel : rendre l'UI plus simple et l'UX plus intuitive en supprimant les sections non nécessaires notamment.

Objectifs

Analyser un site actuel.
Trouver les points de friction.
Imaginer des solutions.
Développer les solutions.

Apports

Analyse d'une page web (référencement / UX-UI).
Mettre en place un cahier des charges.
Création de la maquette grâce à Figma.
Réalisation des pages web (HTML - CSS - JS).
