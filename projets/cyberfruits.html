Description

CyberFruits est mon second projet de partiels de B1. C’est un forum qui permet à quiconque de se libérer concernant le cyber-harcèlement et d'avoir accès à des conseils.

Grâce à ce projet, j'ai pu apprendre énormément de choses sur le PHP. Il m'a permis de me sentir à l'aise avec le langage et de monter une application web dynamique et responsive de toute pièce. C'est la chose sur laquelle je me suis concentré sur le projet. Le design passait après.

Ce projet m'a également permis de m'améliorer dans l'arborescence des fichiers et dans la création/manipulation d'une base de données. Je vous invite à découvrir mon code disponible sur GitHub plus haut.

Objectifs

Concevoir et Réaliser une base de données.
